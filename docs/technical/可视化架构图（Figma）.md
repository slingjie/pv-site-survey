# **Mermaid 图（Figma 原生支持导入）**

你只要复制下面代码，在 Figma → Plugins → Mermaid → Paste → Generate，就能得到一个完整可视化图。

下面是为你生成的 **完整 Figma 版信息架构图（Mermaid）**：

```
flowchart TD

A[踏勘报告 Report] --> B[厂区概况 Plant Overview]
A --> C[建筑屋面 Building Roofs]
A --> D[电气设施 Electrical Facilities]
A --> E[收资清单 Document Collection]

%% 厂区概况
B --> B1[地理位置\n- 工厂地址\n- 当前定位\n- 鸟瞰图上传\n- 房屋位置分布图]
B --> B2[位置交通\n- 交通条件(基本满足/受限)\n- 交通说明\n- 位置分布图上传]
B --> B3[风险区域\n- 风险类型多选\n- 风险说明\n- 风险照片]
B --> B4[用电信息\n- 用电时段\n- 电价类型\n- 电压等级\n- 分时电价\n- 电费单状态\n- 负荷曲线状态\n- 营业执照状态]
B --> B5[产权信息\n- 土地证\n- 产权证/查册信息\n- 业主性质]

%% 建筑屋面
C --> C0[支持多屋顶 Roof-1 Roof-2...]
C --> C1[基础信息\n- 屋顶名称\n- 屋顶面积\n- 屋顶类型\n- 装机容量\n- 鸟瞰图\n- 朝向]
C --> C2[屋顶屋面\n- 利用率\n- 遮挡物类型多选\n- 障碍物照片\n- 污染情况\n- 污染照片]
C --> C3[内部结构\n- 结构类型\n- 结构照片\n- 异常情况多选\n- 异常照片\n- 异常说明]
C --> C4[房屋周边\n- 周边环境照片]

%% 电气设施
D --> D1[现场照片\n- 配电房\n- 变压器\n- 系统图\n- 环境]
D --> D2[设施照片(多设施)\n- 1#设施 2#设施...\n- 配电室\n- 变压器及铭牌\n- 一次系统图\n- 配电柜\n- 计量表\n- 环境]
D --> D3[配电信息\n- 进线电压\n- 计量点电压\n- 开关站\n- 变压器信息列表\n- 位置分布图上传]
D --> D4[并网信息\n- 上网模式\n- 并网柜位置\n- 电缆敷设方式多选\n- 路径说明\n- 接入等级\n- 接入方案描述\n- 注意事项]

%% 收资清单
E --> E1[厂区/厂房资料\n- 总平图\n- 管线图\n- 地勘资料\n- 建筑/结构施工图\n- 屋顶暖通/消防布置图]
E --> E2[资质/产权\n- 营业执照\n- 土地证\n- 房产证]
E --> E3[用电信息(一年)\n- 电费单\n- 负荷曲线]
E --> E4[电气资料\n- 电气总平面\n- 设计说明\n- 高低压一次系统图\n- 接地图\n- 电缆走线图\n- 配电室布置图]
```
