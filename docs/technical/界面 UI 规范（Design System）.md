**完整、专业、可直接进入 UI 设计环节的《界面 UI 规范（Design System）》**

本规范包括：

1. 色彩规范（Color System）
2. 字体与字号规范（Typography）
3. 间距规范（Spacing System）
4. 圆角、描边、阴影规范（Shape & Border）
5. 控件组件规范（Components）
6. 图标尺寸规范（Iconography）
7. 表单、上传组件、标签等核心控件规范

------

# **1. 色彩规范（Color System）**

基于截图抽取的主色系如下：

### **（1）品牌主色 / 强调色**

```
Primary Yellow（主色）: #FFC83D
```

用于：

- 主按钮（保存）
- Tab 选中下划线
- 小 Badge（如 “屋顶-1” 选中）

------

### **（2）辅助色**

```
Secondary Orange（辅助）: #FFA83D（生成踏勘报告按钮描边）
Secondary Blue（链接色）: #4A90E2（示例、选择、跳转箭头）
Green（确认按钮）: #00C853
Red（警告色）: #FF4D4F
```

------

### **（3）中性色（Neutrals）**

```
#000000 / 不透明度 80% ~ 40%（正文、标题、说明文字）
#333333（高对比正文）
#666666（辅助说明）
#999999（弱提示）
#CCCCCC（输入框描边）
#E5E5E5（分割线）
#F5F5F5（背景灰）
#FAFAFA（卡片浅灰背景）
#FFFFFF（白色背景）
```

------

# **2. 字体规范（Typography）**

截图应该是基于 iOS / Android 原生字体渲染风格，推荐使用：

```
iOS：SF Pro Display / SF Pro Text
Android：Roboto
全平台通用 Web/APP：PingFang SC（苹方）
```

### 字号规范（与截图对齐）

| 类型                     | 字号     | 字重    | 示例                   |
| ------------------------ | -------- | ------- | ---------------------- |
| 页面标题                 | 20–22 px | Medium  | “踏勘报告”             |
| 一级表单标题（左侧菜单） | 16 px    | Medium  | “厂区概况 / 建筑屋面”  |
| 分组标题（如“基础信息”） | 15 px    | Medium  | 灰色说明栏             |
| 正文内容（表单标签）     | 14 px    | Regular | “屋顶面积”、“电价类型” |
| 弱提示文字               | 12–13 px | Regular | 上传说明、示例         |
| 按钮文字                 | 16 px    | Medium  | “保存”、“生成踏勘报告” |

行高统一建议：

```
行高 = 字号 * 1.4
```

# **3. 间距规范（Spacing System）**

设计系统采用 4pt 网格体系：

```
xs-4  
sm-8  
md-12  
lg-16  
xl-20  
xxl-24
```

具体落地示例：

- 页面左右边距：16 px
- 卡片内边距：16 px
- 表单项之间垂直间距：12 px
- 分组顶部留白：20 px
- 图片上传卡片：顶部间距约 20–24 px
- 按钮距离底部：16–20 px

此间距策略与您截图里布局贴合度非常高。

------

# **4. 圆角、描边、阴影（Shape & Border & Elevation）**

### **圆角规范（Border Radius）**

```
主输入框、上传框：8 px  
Chip 标签：20 px（半圆）  
按钮：24 px（大圆角按钮风格）  
卡片：12 px  
弹窗：16 px
```

### **描边（Border）**

```
输入框描边：1 px #DDDDDD  
上传虚线框：2 px dashed #CCCCCC  
按钮描边（次按钮）：1.5 px #FFA83D
```

### **阴影（Shadow）**

截图中阴影使用很少（明显偏扁平化），建议：

```
卡片阴影：0 2 6 rgba(0,0,0,0.08)  
弹窗阴影：0 4 12 rgba(0,0,0,0.12)
```

# **5. 控件 / 组件规范（Components）**

下面是你软件复刻时需要重点实现的一套控件规范。

------

## **5.1 输入框（Text Field）**

样式：

- 高度：48 px
- 圆角：8 px
- 背景：#FFFFFF
- 描边：1 px #CCCCCC
- 内侧 padding：左右 12 px
- 右侧可加图标（>）

状态：

- 默认（灰色描边）
- 激活（描边主色 #FFC83D）
- 错误态（描边 red）

------

## **5.2 按钮（Buttons）**

### **主按钮（Primary）**

```
背景：#FFC83D  
文字：#000  
圆角：24px  
高度：48px  
字重：Medium  
```

### **次按钮（Secondary）**

```
背景：透明  
描边：1.5px #FFA83D  
文字：#FFA83D  
圆角：24px  
高度：48px
```

------

## **5.3 标签（Chip Tags）**

用于：风险区域、障碍类型、异常类型等。

样式：

```
高度：32 px  
圆角：20 px  
背景：#F5F5F5  
文字：#333  
内边距：水平 12 px  
间距：8 px
```

选中态：

```
背景：rgba(255,200,61,0.2)
描边：1px #FFC83D
文字：#333
```

------

## **5.4 上传图片卡片（Image Upload Card）**

尺寸：

```
宽度：100%（等比例缩放）
高度：120 px（可自适应）
圆角：8 px  
边框：2 px dashed #CCCCCC  
图标大小：24 px
```

内容居中：

- 上传图标（线性图标）
- “上传/拍照”文字（灰色 #999）

附带右上角：

- 「示例」文本按钮（#4A90E2）

------

## **5.5 下拉弹窗（Bottom Sheet / Picker）**

底部弹窗：

- 圆角顶部：16 px
- 背景：白色
- 选项文字：16 px
- 行高：48 px
- 底部按钮：取消（灰）、确定（绿色）

按钮样式：

- 取消按钮背景：#F5F5F5
- 取消文字：#333
- 确定按钮背景：#00C853
- 确定文字：白色

------

## **5.6 单选（Radio）与多选（Checkbox）**

### Radio（如：分时电价 是/否）

```
直径：18 px  
颜色：#FFC83D  
```

### Checkbox（用于电缆敷设方式）

```
尺寸：20 px  
描边：#999  
选中背景：#FFC83D  
勾线：白色
```

------

## **5.7 Tab（顶部一级导航）**

- 高度：48 px
- 文字：14–15 px
- 选中态：主色下划线（2px #FFC83D）
- 未选中：#666

------

# **6. 图标规范（Iconography）**

从截图可提炼如下图标系统：

```
线性图标（Outlined）
尺寸：20–24 px
颜色：#666 或 #999
圆形动作图标：24–28px（右上角菜单）
```

建议选型：

- 阿里 iconfont（默认即可）
- 或 Material Symbols Outlined
